Install Instructions

  1. Download the installer for your platform from this page.
  2. Windows: run the .exe and follow setup prompts.
  3. Linux: run sudo apt install ./streamflix_2.3.0_amd64.deb.
  4. Launch StreamFlix from the app menu or desktop shortcut.

Release Details
